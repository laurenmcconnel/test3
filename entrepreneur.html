The Feasibility of Twitter Sentiment Analysis Implimentation

For this project, I used various data cleaning and feature engineering techniques in Python on an unseen dataset.

Data Cleaning

For this section, I counted and removed null values, along with identifying the differences in interval and ordinal categorical variables, and how they differ when converted to a numerical representation. I also explained the reasons for using interval or ordinal techniques.

Feature Engineering

For this section, using Supervised Machine Learning Teachniques I created a train and test dataset. I then tested the accuracy of the model using of SVM and RandomForests and accuracy score.
